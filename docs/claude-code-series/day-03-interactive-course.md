### Recommended Learning Path (from Daniel Avila)

#### Phase 1: Foundations
1. Installation and setup
2. Authentication methods
3. Basic commands
4. Permission system
5. Git integration

#### Phase 2: Core Skills
1. CLAUDE.md configuration
2. Context management
3. Prompt engineering for code
4. File operations
5. Code review workflows

#### Phase 3: Advanced Techniques
1. Custom slash commands
2. MCP server integration
3. Subagents
4. Automation workflows
5. Team collaboration

### Learning Philosophy
- Claude Code isn't about replacing coding skills
- It's about removing friction from development
- Remembers context you forget
- Spots patterns you miss
- Handles routine tasks

### Key Concepts to Master

1. **Interactive Mode**
   - Starting sessions
   - Conversation flow
   - Context preservation

2. **One-Shot Mode**
   - Quick queries
   - Scripting integration
   - Pipeline usage

3. **Plan Mode**
   - SHIFT+TAB to enter
   - Research without changes
   - Strategic thinking

4. **Session Management**
   - `/resume` for continuity
   - `/clear` for fresh starts
   - Session history

### Practice Exercises to Include

1. **Beginner**: Create a simple function with Claude's help
2. **Intermediate**: Refactor existing code using Claude
3. **Advanced**: Build a complete feature end-to-end

### Common Learning Mistakes
- Giving vague instructions
- Not providing enough context
- Ignoring Claude's questions
- Over-relying on AI for simple tasks
- Not reviewing generated code

## Hands-On Project Ideas

### Project 1: Hello Claude (Beginner)
```
Goal: Create a simple CLI tool with Claude's help
Tasks:
- Initialize a new Node.js project
- Ask Claude to create a "hello world" CLI
- Add argument parsing
- Add color output
```

### Project 2: Code Reviewer (Intermediate)
```
Goal: Use Claude to review and improve existing code
Tasks:
- Find a messy file in your codebase
- Ask Claude to identify issues
- Implement suggestions one by one
- Compare before/after
```

### Project 3: Feature Builder (Advanced)
```
Goal: Build a complete feature from scratch
Tasks:
- Describe the feature to Claude
- Let Claude plan the implementation
- Review and refine the plan
- Implement step by step
- Write tests with Claude's help
```
