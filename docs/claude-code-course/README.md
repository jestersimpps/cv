A comprehensive course taking you from beginner to expert in Claude Code.

## Prerequisites

- Node.js 18.17.0 or higher
- An Anthropic account with billing enabled
- Basic command line familiarity
- A code editor (VS Code recommended)

## Quick Start

If you're completely new, start with Chapter 1. If you already have Claude Code installed:

- **Know the basics?** Jump to Chapter 3 (Project Configuration)
- **Want power user tips?** Skip to Chapter 9 (Power User Secrets)
- **Setting up for a team?** Focus on Chapters 7-8
